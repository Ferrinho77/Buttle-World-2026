# Fanta World Cup

App mobile/web per creare leghe private, invitare partecipanti via email, inserire pronostici e calcolare classifiche.

## Setup

1. Copia `.env.example` in `.env`.
2. Inserisci `VITE_SUPABASE_URL` e `VITE_SUPABASE_ANON_KEY`.
3. Esegui `supabase/schema.sql` in Supabase SQL Editor.
4. Pubblica su Vercel.
